This page displays all glossary entries of the current project. Every entry of the glossary should be evaluated strictly and should be kept up to date. Please communicate all definitions within all teams.

Please use the following conventions for the glossary (in German):

  • Glossarsprache ist deutsch
  • Glossareinträge werden in "natürlicher und verständlicher Sprache" definiert (so dass sie von Außenstehenden verstanden werden können)
  • Schreibweise ist so, wie man auch in einem Textdokument schreiben würde, also:
    • nicht EnttaeuschendeNachricht_am_Mittag
    • sondern enttäuschende Nachricht am Mittag
